# COVID-19-dashboard
>Covid-19 Dashboard for India

## Built with
* Spring Boot

## Other technologies/frameworks used
* Spring Security
* Spring JPA
* Thymeleaf
* chart.js
* DataTables

## Dependencies
* [https://api.covid19india.org/data.json](https://api.covid19india.org/data.json)
